# How to run ?

## TCP:
- Server:
```
gcc my_tcpserver.c -o my_tcpserver
./my_tcpserver
```

- Client:
```
gcc my_tcpclient.c -o my_tcpclient 
./my_tcpclient <YOUR_FILENAME.EXT>
```

## UDP: 
- Server:
```
gcc my_udpserver.c -o my_udpserver
./my_udpserver
```

- Client:
```
gcc my_udpclient.c -o my_udpclient 
./my_udpclient <YOUR_FILENAME.EXT>
```
